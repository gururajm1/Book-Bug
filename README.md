# First Steps

### git clone (`https://github.com/gururajm1/Book-Bug.git`)

### `cd client`

Navigate to the Client Directory

### `npm install`

Install Node Packages

### `npm start`

Start the client 


### `cd server`

Navigate to the Server Directory

### `npm install`

Install Node Packages

### `nodemon server.js`

Start the server js
